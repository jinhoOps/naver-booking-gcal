# Naver Booking → Google Calendar Helper

네이버 예약은 기본적으로 구글 캘린더 연동 기능을 제공하지 않습니다.  
이 확장프로그램은 네이버 예약 상세 페이지에서 예약 정보를 자동으로 추출하여  
원클릭으로 Google Calendar 일정 생성 화면을 열어주는 도구입니다.

## 주요 기능

- 네이버 예약 상세/공유 페이지 자동 감지
- 예약 제목, 담당자, 메뉴, 일시, 주소 자동 추출
- 원본 예약 링크 자동 포함
- Google Calendar 일정 작성 페이지 자동 오픈
- 깔끔한 플로팅 버튼 + 확인 팝업 UI

## 설치 방법 (Chrome 개발자 모드)
" 언젠가는 크롬웹스트어에 올려야지..
1. 이 저장소 전체를 다운로드(zip) 하거나 Git clone 합니다.
2. Chrome 주소창에 아래 입력  
   `chrome://extensions`
3. 오른쪽 상단 개발자 모드 활성화
4. 왼쪽 상단 압축 해제된 확장프로그램을 로드 클릭
5. 프로젝트 폴더 선택
6. 네이버 예약 상세 페이지를 열면 우하단에  
   "캘린더 추가" 버튼이 나타납니다.


## 사용 방법

1. 이 확장프로그램을 설치합니다.
2. 네이버 예약 상세 페이지 또는 공유 페이지를 열면  
   우하단에 "캘린더 추가" 버튼이 자동으로 나타납니다.
3. 버튼 클릭 → 예약 정보 확인 팝업 → "추가" 선택
4. Google Calendar 일정 생성 페이지가 열립니다.

## 지원 페이지 예시

- https://booking.naver.com/my/share/bookings/XXXXXXXX
- https://booking.naver.com/my/bookings/XXXXXXXX

## 제한 사항

- Google Calendar API 직접 호출 기능은 포함되지 않습니다.  
  (알림 시간 등은 캘린더 기본 설정을 따릅니다.)
- 네이버 예약 페이지 구조 변경 시 정보 추출 방식이 조정될 수 있습니다.

## 목적

네이버 예약 일정은 구글 캘린더에 연동이 불가합니다.
매번 수동으로 복사·붙여넣기 하던 과정을 간단화하기 위해 만든
가벼운 기능 확장 프로그램입니다.
